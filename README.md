# flight-data-analysis
Python + SQL analysis for aviation flight data
This project demonstrates how to use python and SQL to process and anlyze large-scale aviation data.

## Features
- Connect to relational database (MySQL/PostgreSQL)
- Query flight data using SQL
- Analyze flight operations with pandas
- Support large datasets (thousands of flights per day)

## Structure
-'sql/':SQL query examples
-'src/':Pyhton sccripts for database connection and analysis
-'notebooks/':Exploratory data analysis

## Application
This project is designed for aviation operation and environmental research, such as aircraft-scale analysis.
